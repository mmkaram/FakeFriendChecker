# Fake Friend Checker
> Lots of rats nowadays, stay safe

This is a python script that compares your follower and following lists based off of exported data from Meta.

Instructions:
- Go to your Meta account center and click on the "Your information and permissions" tab
- Click on "Export your information" and select json as the file type
- Save the followers.json and following.json into the same directory as main.py
- run `python3 main.py`
